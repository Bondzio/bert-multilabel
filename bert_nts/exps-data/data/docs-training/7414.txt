AmphiMove
Es wird davon ausgegangen, dass die Anwendung von Pflanzenschutzmitteln eine Risiko für Amphibien bedeutet. Um das Risiko bewerten zu können sind Daten zur Habitatpräferenz, und Migration von Amphibien notwendig, fehlen aber. Die notwendigen Daten dafür, sollen im Vorhaben erhoben werden.
Die Belastung durch Fang, Markierung und Telemetrie wird als gering angesehen. Die Tiere bleiben in der Natur.
Das Verhalten der Tiere muss unter natürlichen Bedingungen untersucht werden. Nur so können Aussagen über Habitatpräferenz und Migrationsmuster getroffen werden. Eine Freilandstudie ist dafür unerlässlich. 
Die Zahl der notwendigen Tiere wurde so ermittelt, dass eine statistische Aussage getroffen werden kann.  
Es werden keine Schäden erwartet. Ein Versuch ist nur mit den betroffenen Tierarten möglich.
